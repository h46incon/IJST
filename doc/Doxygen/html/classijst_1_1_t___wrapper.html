Detailed Description

template<typename _T>
class ijst::T_Wrapper< _T >

Common wrapper to hold object via a standard-layout object.

offset() macro requires the object is standard-layout, * i.e, all members in the object is standard-layout. This wrapper could wrap any type to a standard-layout type.
