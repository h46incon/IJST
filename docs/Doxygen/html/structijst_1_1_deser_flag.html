◆ kMoveFromIntermediateDoc

const Flag ijst::DeserFlag::kMoveFromIntermediateDoc = 0x00000008
static

Set if move resource (to unknown or T_raw fields, copy by default) from intermediate document when deserialize.

This option will speed up deserialization. But the allocator document will not free before the deserialized object destroyed. And the nested object will use parent's allocator. Be careful when moving the nested object to another object (e.g, calling RValue copy constructor), because the parent will free allocator when destroy.

Note
Using this option carefully
