◆ IJST_EXTERN_TEMPLATE

#define IJST_EXTERN_TEMPLATE   0

ijst support extern template to speed up compilation. To enable this option, set the macro to 1 before including ijst headers. User should declare IJST_EXPLICIT_TEMPLATE in one of cpp files to instantiation the templates.

Note
require extern template support (since C++11) to enable this option
