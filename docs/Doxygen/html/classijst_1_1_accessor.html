Detailed Description

template<typename Encoding = rapidjson::UTF8<>>
class ijst::Accessor< Encoding >

Accessor of ijst struct.

Template Parameters
EncodingEncoding for json struct

User can access and modify fields, serialize and deserialize of a structure via it.

Move deserialize form json document.

Deserialize from json document. The source object may be stolen after deserialize. Because the accessor need manager the input allocator, but the Allocator class has no Swap() interface, so use document object instead.

Parameters
srcDocStolenInput document object
deserFlagDeserialization options, options can be combined by bitwise OR operator (|)
pErrDocOutError message output. Null if do not need error message
Returns
Error code
Note
Handler allocator VERY CAREFULLY. see DeserFlag::kMoveFromIntermediateDoc.
It will free own allocator
The source document may be changed after deserialize
Make sure srcDocStolen use own allocator, or use allocator in this object
It will not copy const string reference in source json. Be careful if handler such situation, e.g, json object is generated by ParseInsitu().

DeserFlag::kMoveFromIntermediateDoc
